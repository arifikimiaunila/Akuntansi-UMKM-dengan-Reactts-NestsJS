Frontend React + TypeScript untuk dashboard payroll dan akuntansi.

Quick start:

1. `cd react-frontend`
2. `npm install`
3. `npm run dev`

Fitur:

- login JWT (owner/admin/finance/hr)
- pembatasan fitur berdasarkan role (RBAC)
- tambah karyawan
- proses payroll per periode
- posting jurnal manual
- monitoring metrik biaya dan payroll
- laporan trial balance, laba-rugi, dan neraca

Contoh akun login:

- owner / Owner#123
- admin / Admin#123
- finance / Finance#123
- hr / Hr#12345
